# JARVIS Desktop M1 Demo

This is the Milestone 1 interactive fake flow demo.

It replays contract-shaped events to validate UX before real sidecars are integrated.

## Run

```bash
npm install
npm run dev
```

## What it demonstrates

- General/Code mode switch
- Token-style streaming response
- Permission request and approve/deny flow
- Run card rendering (code mode)
- Audit log updates
- Scripted demo scenarios (left sidebar) for repeatable Milestone 1 demos
